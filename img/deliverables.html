Bootstrap Features

Bootstrap Features Used on This Page

Below are the Bootstrap features implemented in the webpage:

1. Grid System:

  • Responsive Grid: The page uses Bootstrap's grid system with classes like `col-lg-6`, `col-md-6` to organize content for different screen sizes.
  • Example: Inside the `#projects` section, the project cards are placed in columns using `
    `.

2. Navbar:

  • Responsive Navigation: A collapsible navigation bar is created using `navbar`, `navbar-toggler`, and `navbar-collapse` classes, making it mobile-friendly.
  • Example: The navigation bar in the header adapts to smaller screen sizes by collapsing into a hamburger menu.

3. Buttons:

  • Styled Buttons: Bootstrap button classes like `btn btn-success` are used to style buttons and links.
  • Example: Buttons in the education section to select the year or GitHub links are styled with `btn btn-success`.

4. Typography:

  • Text Alignment and Font Styling: Classes like `text-center`, `text-primary`, and `font-weight-bold` are used for aligning text and styling headers.
  • Example: The text in the `#projects` section and the dynamic skill display uses these typography classes.

5. Spacing and Padding:

  • Margin and Padding Classes: Classes like `mt-1`, `mb-4`, `gy-5`, `pt-4`, `pt-lg-0` are used for controlling spacing between elements.
  • Example: Elements in the sections have margins and paddings for proper spacing.

6. Cards:

  • Card Component: Bootstrap's card component is used to display content like project descriptions in a structured format with images and text.
  • Example: The project cards in the `#projects` section use `
    ` for styling.

7. Icons:

  • Font Awesome Icons: Social media icons such as Twitter, LinkedIn, and Instagram are added using Font Awesome icons.
  • Example: Icons are used in the navigation bar and contact section for easy access to social media.

8. Collapse:

  • Collapsible Content: The navbar menu collapses into a hamburger menu on smaller screens using the `collapse` component.
  • Example: The `navbar-collapse` class is used to hide and show the menu items on smaller devices.

9. Modal or Dynamic Content with jQuery:

  • Dynamic Skill Change: The dynamic skills section uses jQuery to change the displayed skill every few seconds.
  • Example: The skill set in the `#skills` section is updated dynamically with `fadeOut` and `fadeIn` effects from jQuery.

10. Utilities:

  • Text and Background Color Utilities: The `bg-light`, `bg-gray`, and `text-center` classes are used for controlling background and text color.
  • Visibility Utility: The `.d-none` class is used to hide and show the year sections in the Education section.

11. Responsive Images:

  • Responsive Images: Although not explicitly in the code, the `img-fluid` class can be used to make images responsive, allowing them to scale according to the screen size.
  • Example: The images in the `#home` and `#projects` sections could be made responsive by adding `img-fluid` to the `` tags.
